# DataTalk: Talk with your Database!

**Talk to Your Database: Natural Language SQL Query Interface**

This project allows users to interact with a PostgreSQL database containing information about countries using natural language queries. It leverages the power of AI to translate English questions into SQL queries, making database interactions more intuitive and accessible to the general public.

**Major Achievements**

- **Natural Language Processing:** Utilizes Google's Generative AI (Gemini 1.5 Flash model) to interpret user queries in plain English and convert them to SQL.
  
- **Real-time Database Interaction:** Connects to a PostgreSQL database and executes dynamically generated SQL queries based on user input.
  
- **User-friendly Interface:** Provides a clean, responsive web interface for users to input queries and view results.

**Technologies Used**

- **Frontend:** React.js, CSS3
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **AI:** Google Generative AI (Gemini 1.5 Flash)

**How to Use**

Enter your question about countries in the search bar. For example:

- "How many countries are there in continent Asia?"
- "What are the top 5 countries with the most population?"
- "List all countries with a life expectancy above 80 years."

Click the "Submit" button or press Enter.
View the results displayed in a formatted JSON structure.
